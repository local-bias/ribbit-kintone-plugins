# AGENTS.md

- パッケージ管理はpnpmを使用する。
- コードはTypeScriptで記述する。
- コードフォーマットにはbiomeを使用する。

## スキル

詳細なドメイン知識は必要に応じて参照すること。

| スキル | 内容 |
|--------|------|
| `kintone-api` | kintoneグローバル変数・REST API・外部APIのリクエストパターン |
| `kintone-plugin` | プラグイン構造（設定画面/操作画面）・ストレージ |
| `react-kintone` | React状態管理（jotai）・スタイリング（emotion）・コンポーネント配置 |
| `kintone-file-download` | kintone添付ファイルのダウンロード実装 |
| `kintone-styling` | kintone標準スタイルとの競合対策・スタイリング注意点 |
