# 📦 添付ファイル一括ダウンロードプラグイン

kintoneの添付ファイルをZIPファイルにまとめて一括ダウンロードするプラグインです。

## 機能

- 複数の添付ファイルフィールドに対応
- サブテーブル内の添付ファイルフィールドにも対応
- 3箇所にダウンロードボタンを設置可能：
  - **一覧画面のヘッダー**: 現在絞り込まれているすべてのレコードの添付ファイルを一括ダウンロード
  - **レコード詳細画面のヘッダー**: 表示中のレコードの添付ファイルをダウンロード
  - **スペースフィールド**: 指定したスペースフィールドにダウンロードボタンを配置

[ホームページ](https://ribbit.konomi.app)

## 🔧 使い方

### パッケージのインストール

```
npm install
```

### プラグインの秘密キーを作成と、SSL 証明書の作成

```
npm run init
```

### ご利用の Kintone へアップロード + ファイルの変更を監視

```
npm run dev
```

### リリース用 zip ファイルの生成

```
npm run build
```
