# 👀 他画面表示プラグイン

特定のアクションをトリガーに、他のアプリやウェブページを表示するプラグインです。

特定のアプリを閲覧、編集しながら、別のアプリやウェブページを参照したい場合に便利です。

## 機能

アプリの一覧画面、詳細画面、編集画面にボタンを設置し、ボタンをクリックした際に、指定したアプリやウェブページを表示します。

表示モードに応じたコンポーネントにiframeで表示します。

## カスタム項目

- ページモード
  - 他アプリを開くのか、特定のURLを開くのかを選択します。
- 表示画面
  - 一覧画面、詳細画面、編集画面のどの画面にボタンを設置するかを選択します。複数選択可能です。
- ボタンラベル
  - ボタンに表示するラベルを設定します。
- 表示モード
  - どのような形式でページを表示するのかを選択します。
    - モーダル
    - ドロワー
    - 分割表示
- URL
  - ページモードで「URL」を選択した場合に表示されます。開きたいURLを入力します。
- アプリ
  - ページモードで「他アプリ」を選択した場合に表示されます。開きたいアプリを選択します。

## ユースケース

- 営業支援アプリを閲覧しながら、顧客管理アプリを参照したい
- 顧客管理アプリを登録する際に、郵便番号から住所を検索する外部サイトを参照したい
- プロジェクト管理アプリを編集する際に、関連するタスク管理アプリを参照したい
- 見積アプリを閲覧する際に、製品マスタアプリを参照したい
- 顧客管理アプリを閲覧する際に、会社のポータルサイトを参照したい
