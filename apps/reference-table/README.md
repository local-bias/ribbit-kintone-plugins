# 関連レコード拡張テーブルプラグイン

[ホームページ](https://ribbit.konomi.app)

既存の関連レコードフィールドを置き換え、関連先アプリのレコードを検索・絞り込み・集計できるテーブルとして表示する kintone プラグインです。関連先アプリにサブテーブルがある場合は、必要に応じてサブテーブル行も同じテーブルに展開できます。

## 設定項目

- 表示スペース: このプラグインの表示先となるスペースフィールド
- 関連先アプリ: 関連レコードとして参照するアプリ
- 照合フィールド: このアプリの値と関連先アプリの値を比較するフィールド
- 表示する関連レコードフィールド: 関連先アプリからテーブルに表示するフィールド
- 表示するサブテーブル: 関連先アプリ内のサブテーブル。使用しない場合は未設定のまま利用できます
- サブテーブルから表示する列: サブテーブルを使用する場合にテーブルへ表示するサブテーブル内フィールド
- 一度に表示するレコード数: ページングで一度に表示する関連レコード数
- 集計値の端数処理: 集計値を表示するときの四捨五入・切り上げ・切り捨て
- 集計値の小数点以下最大表示桁数: 集計値の小数点以下を表示する最大桁数
- 並び順: 関連レコード単位の表示順

## 🔧 使い方

### パッケージのインストール

```
npm install
```

### プラグインの秘密キーを作成と、SSL 証明書の作成

```
npm run init
```

### ご利用の Kintone へアップロード + ファイルの変更を監視

```
npm run dev
```

### リリース用 zip ファイルの生成

```
npm run build
```
