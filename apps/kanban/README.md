# カンバンボード kintone プラグイン

kintone のカスタムビューにカンバンボードを表示するプラグインです。

## 機能

- ドロップダウンやラジオボタン、プロセス管理のステータスフィールドをカラムとして表示
- ドラッグ＆ドロップによるカード移動（ステータス変更）
- カードの追加・編集・削除
- カードの検索フィルタリング
- カラムの色設定
- WIP（仕掛品）制限の設定
- 担当者アバター・説明文のカード表示
- 多言語対応（日本語・英語）

## 設定項目

| 項目                     | 必須 | 説明                                     |
| ------------------------ | ---- | ---------------------------------------- |
| 対象ビュー               | ✅   | カスタムビューを選択                     |
| カード名フィールド       | ✅   | カードのタイトルとして表示するフィールド |
| ステータスフィールド     | ✅   | カラムの分類に使用するフィールド         |
| 担当者フィールド         | -    | カードに担当者アバターを表示             |
| 説明文フィールド         | -    | カードに説明文を表示                     |
| カラム色設定             | -    | 各ステータスのカラムヘッダー色           |
| カード追加表示フィールド | -    | カードに追加で表示するフィールド         |
| WIP制限                  | -    | 1列あたりの最大カード数                  |

## 開発

```bash
pnpm install
pnpm dev
```

## ビルド

```bash
pnpm build
```
